# SLICE Logo Assets

Drop this `slice-assets` folder inside your website or app `assets/` directory.

## Recommended files

### Website logo
Use:
- `logo/slice-logo-mark-512.png`
- `logo/slice-logo-mark-1024.png`

### Favicon / browser tab icon
Use:
- `favicon/favicon.ico`
- `favicon/favicon-32x32.png`
- `favicon/apple-touch-icon.png`
- `site.webmanifest`

Example HTML:

```html
<link rel="icon" href="/assets/slice-assets/favicon/favicon.ico" sizes="any">
<link rel="icon" type="image/png" href="/assets/slice-assets/favicon/favicon-32x32.png">
<link rel="apple-touch-icon" href="/assets/slice-assets/favicon/apple-touch-icon.png">
<link rel="manifest" href="/assets/slice-assets/site.webmanifest">
<meta name="theme-color" content="#ff6b00">
```

### Mobile app icons
Use:
- iOS: `app-icons/ios/ios-icon-1024x1024.png` for App Store source icon
- Android: `app-icons/android/android-icon-512x512.png`
- Android/PWA maskable: `app-icons/android/maskable-icon-512x512.png`

### Social sharing image
Use:
- `social/og-image-1200x630.png`

## Brand colors

- Orange: `#ff6b00`
- Dark navy: `#061524`
- White: `#ffffff`
